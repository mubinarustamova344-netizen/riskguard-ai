Project Overview

RiskGuard AI is a practice-based capstone project for the Pearson BTEC Level 6 Diploma in Digital Technologies (Unit 2: Independent Project). It addresses the industry challenge of accurate, data-driven risk stratification in motor insurance underwriting.

The project follows the full ML lifecycle: problem identification → data generation → feature engineering → model training and evaluation → production deployment as a Flask web application with an interactive UI and REST API.

Four classification algorithms were trained, cross-validated, and compared. The best model (Gradient Boosting, ~86% test accuracy) is served in real-time, with full explainability through factor contribution charts and personalised recommendations.
